# DC ELO v3 — Rating Dashboard

Next.js 15 app ready for Vercel. Data from Google Sheets (no API key needed).

## Quick start
```bash
npm install && npm run dev
```

## Deploy to Vercel
1. Push to GitHub
2. Import on vercel.com → auto-detects Next.js → Deploy
3. No environment variables needed

## New in v3
- Player Comparison (radar chart, stat bars)
- Articles section with categories
- Organization page with team members
- CS / EN / FR translations
- Dark + Light mode
- Mobile sidebar
